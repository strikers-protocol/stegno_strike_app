### Basic Procedures

#### 1. ENCODE Mission (Hiding Data)
* **A. Select Source:** Choose a `.png` or `.jpg` cover image.
* **B. Input Payload:** Type or paste your secret message (text/passwords).
* **C. Inject Key:** Set a strong encryption key. *Crucial: If you forget this key, the data is lost forever.*
* **D. Execute:** Click the 'ENCODE' button to save the new image.

#### 2. DECODE Mission (Extracting Data)
* **A. Select Target:** Choose the image that contains hidden data.
* **B. Inject Key:** Enter the exact encryption key used during encoding.
* **C. Execute:** Click the 'DECODE' button. The hidden message will appear in the payload box.
